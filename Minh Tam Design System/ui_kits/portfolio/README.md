# Portfolio UI Kit

A recreation of the Minh Tam homepage — dark editorial, violet-accented.

## Components
- `HeroSection.jsx` — grid background + clip-reveal headline + metric panel + proof chips + dual CTA
- `Navbar.jsx` — floating glass pill with pulse dot
- `WorksSection.jsx` — 2-col staggered grid of work cards with hover states
- `ProcessSection.jsx` — 3-step centered timeline with ghost numerals
- `CtaSection.jsx` — centered headline + violet pill CTA
- `Footer.jsx` — word-mark + social glyphs

## Notes
This is a cosmetic recreation. Real portfolio uses framer-motion for clip-reveal and scroll-in; we approximate with CSS transitions.
